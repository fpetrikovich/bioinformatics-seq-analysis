# bioinformatics-seq-analysis
Analysis of an unknown sequence.

## Instalar

Crear el entorno virtual con `virtualenv`:
```
virtualenv .env
o
python3 -m venv "virtualenv"
```

Activar el entorno:
```
source .env/bin/activate
```

Instalar dependencias:
```
pip install -r requirements.txt
```
